Exercices structure et terminologies

Question 1 : La déclaration DOCTYPE est-elle obligatoire ? Où doit-elle être placée ? Quelles informations contient-elle ?

Réponse 1 : Oui, elle est obligatoire; Elle est placé au début du code HTML; !DOCTYPE HTML informe le navigateur web que le document est de type html et qu'il s'agit (ici) de la derniére version de la langage, HTML5

Question 2 : c'est quoi un élément dans le langage HTML ?

Réponse 2 : Les éléments en HTML : ils définissent la structure et le conteneu des objets au sein d'une page, c'est la balise avec son contenu.

Question 3 : c'est quoi une balise dans le langage HTML ?

Réponse 3 : Balises : elles sont reconnaissables car elles sont encadrées d'un cheveron ouvrant et d'un cheveron fermant. L'ouverture d'une balise détermine le début d'un élément.
Sa fermeture détermine sa fin. Le contenu lui est insére entre une balise ouvrante et fermante représente le contenu de cet élément. Par exemple, un lien html sera encadré d'une balise A ouvrante et fermante. Il existe aussi des balises auto-fermante(aussi dites orpholine), cesbalises correspondent à des éléments pinctuels dans une page html, pas besion de fournir une information de fin (insérer une image ou un lien par exemple). les plus courantes:
<br/>
<img/>
<met/>
<input/>
<hr/>
<link/>

Question 4 : c'est quoi un attribut dans le langage HTML ?

Réponse 4 : Attributs : ce sont des propriétés utilisées pour fournir des informations complémentaires sur un élément. les attributs les plus communs incluent l'attribut ID (qui identifie un élement), l'attribut CLASS (qui classifie un élément ), l'attribut SRC(qui spécifie la source d'un contenu qui peut être imbriqué) et l'attribut HREF (qui contient un lien hypertexte vers une ressource liée). les attributs sont définis au sein de la balise ouvrante aprés le nom de l'élément:
<a href="http://github.com/Ma6Tvacoder-Docs/partages">lien</a>

Question 7 : Où est déclaré le nom de l'élément racine du document ? Quel est son rôle et son nom ?

Réponse 7 : il est déclaré après Doctype , son nom c'est <html> et son rôle est de déclarer la langue utilisé sur notre document.

Question 8 : Quels sont les éléments enfants de l'élément HTML ?

Réponse 8 : head et body sont les élément enfants de l'élément html
Tous les fichiers html ont une structure de base qui implique de décaler les éléments suivants : 

                                        <!DOCTYPE html> 

                                        <html>

                                            <head>

                                            </head>

                                            <body>

                                            </body>

                                        </html>

Question 9 : À quoi sert l'élement <head> ?

Réponse 9 : c'est l'élément où on définit tout les informations générale sur la page, la balise head identifie le début du document et inclut les métadonnées(informations sur la page). Le contenu à l'intérieur de la balise head n'est pas affiché par le navigateur puisq'il donne des informations sur la page(liens vers des fichiers externes, utilisation du language html, titre de la page.

Question 10 : Quel élement est obligatoire dans l'élément <head> ? À quoi sert-il ?

Réponse 10 : l'élément obligatoire c'est<title>. : définit le titre du document (qui est affiché dans la barre de titre du navigateur ou dans l'onglet de la page)

Question 14 : Quel est le rôle de l'élément <base>

Réponse 14 : elle spécifie une adresse par défaut pour tous les liens sur une page, pour déclarer un chemin absolu, elle se met à l'intérieur de l'élément head.
