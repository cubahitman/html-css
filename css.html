Le CSS, Cascading Style Sheet

Le CSS permet de faire du style sur nos pages html.

On peut placer le CSS a trois endroits.

Le CSS dans la balise

La première façon de faire du style dans un élément est de déclarer l'attribut style sur ceui-là. Ce n'est pas la meilleure manière de le faire.

COucou, je suis en rouge.

Le CSS dans l'en tête

Une seconde façon de faire du style sur notre page est de mettre la balise style dans la partie head. Cette méthode n'est toujours pas la plus conseillée mais elle permet plus de choses que la méthode balise. Exemple, on met du rose en fond sur le h1 et sur les h2

CSS dans une feuille de style externe

La dernière et meilleure méthode pour faire du style est la méthode suivante : on crée un dossier css avec un fichier style.css à l'intérieur. Il faudra alors lier nos deux fichiers grâce à la balise link. On pourra alors faire du style sur les éléments qui se trouvent dans notre page. Exemple, on crée un fichier style.css dans lequel on déclare que tous les paragraphe doivent être en vert.
